### Each Solve Task returns an URL with task id.

*** method: `Get`***
```
"https://free.nocaptchaai.com/status?id=h-q7FBc9fXJxxxx"

```

## Sample Response

```json
{
    "createdat": 1662353086,
    "id": "h-q7FBc9fXJ0V69ox4",
    "status": "new",
    "target": "airplane",
    "url": "https://free.nocaptchaai.com/status?id=h-q7FBc9fXJxxxx"
}

```
